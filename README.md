# 🔴 CRIMEVIS AI
### Real-Time Crime Detection & Intelligent Threat Response System

## 🎯 What is CRIMEVIS AI?
CRIMEVIS AI is a real-time intelligent surveillance system 
that uses computer vision and deep learning to monitor CCTV 
feeds, detect threats, and automatically alert authorities.

## ✨ Features Built
- 🎥 Live camera feed monitoring
- 🤖 Person detection using YOLOv8
- 🔪 Weapon detection (knife, scissors)
- 🥊 Fight and aggression detection
- ⏱️ Loitering detection (30 sec threshold)
- 📊 Threat score system (1-10)
- 📸 Auto screenshot on threat detection
- 📧 Email alerts with screenshot attached
- 🌐 Live web dashboard (Flask)
- 📄 PDF incident report generation

## 🔮 Upcoming Features
- 🫀 Fall and unconscious detection
- 👋 Silent SOS gesture recognition
- 😨 Facial distress detection
- 🎤 Audio scream detection
- 🚔 Auto police and ambulance SMS alerts

## 🛠 Tech Stack
- Python 3.13
- OpenCV
- YOLOv8 (Ultralytics)
- MediaPipe
- Flask
- ReportLab
- SMTP Email

## 🚀 How To Run

### Install libraries
pip3 install opencv-python ultralytics flask reportlab

### Run detection
python3 main.py

### Run dashboard
python3 dashboard.py
Open browser → http://localhost:8080

### Generate PDF report
python3 pdf_report.py

## 📁 Project Files
- main.py → Main detection system
- dashboard.py → Live web dashboard
- email_alert.py → Email alert system
- pdf_report.py → PDF report generator
- incidents/ → Auto saved screenshots
